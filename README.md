# Sales Data Cleaning Project

## Objective
To clean and prepare raw sales data for analysis.

## Steps Performed
- Data understanding and data dictionary creation
- Data quality assessment
- Data cleaning:
  - Removed unnecessary time from date column
  - Standardized text values
  - Verified numeric columns
  - Created Profit_Type column

## Tools Used
- Excel

## Outcome
A clean and analysis-ready dataset.
