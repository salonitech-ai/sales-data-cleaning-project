# Sales Data Analysis Dashboard

## 📊 Project Overview
This project involves cleaning, analyzing, and visualizing sales data using Excel and Power BI.

## 🛠 Tools Used
- Excel (Data Cleaning)
- Power BI (Dashboard)

## 📈 Key KPIs
- Total Sales: ₹2.7M
- Total Profit: ₹425K
- Total Orders: 100
- Profit %: 15.75%

## 🔍 Insights
- Mumbai has the highest sales
- Chennai has the lowest sales
- Headphones generate the highest profit
- 71% transactions are profitable
